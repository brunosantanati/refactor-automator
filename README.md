#### Program arguments  
<code><GITHUB_TOKEN> <GROUP_ID> <ARTIFACT_ID> <NEW_VERSION> <REPO_1> <REPO_2></code>  
Sample:  
<code>ghp_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx org.apache.commons commons-lang3 3.19.0 brunosantanati/demo1 brunosantanati/demo2</code>  

#### Environment variable
It's necessary to configure both environment variables: `JAVA_HOME` and `MAVEN_HOME`. In case you are running this application from an IDE, you should configure these env variables within your IDE.
